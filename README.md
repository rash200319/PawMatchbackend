# PawMatch 🐾

**Smarter Adoptions, Happier Endings**


PawMatch is a next-generation pet adoption platform designed to prioritize compatibility between pets and adopters. By moving beyond simple listings and integrating a lifestyle-based matching algorithm ("Pawsonality"), we aim to reduce return rates and ensure every animal finds a forever home where they can thrive.

## 🌐 Live Site

Check out the deployed application: **[https://pawmatch.dpdns.org/](https://pawmatch.dpdns.org/)**

## 🚀 Key Features

-   **🧠 Compatibility Engine:** A weighted scoring algorithm that matches pets to adopters based on lifestyle, activity level, and living situation.
-   **🛡️ Identity Verification:** Prevents fraud with document verification for shelters.
-   **🐕 Buddy-Check System:** Analyzes temperament compatibility with existing pets in the household.
-   **📅 Welfare Tracker:** Automated 14-day post-adoption monitoring based on the 3-3-3 rule.
-   **🏠 Foster-to-Adopt:** A seamless pathway for users to foster pets before committing to adoption.
-   **⚡ Community Reports:** Crowdsourced tool for reporting injured or abandoned animals.

## 🛠️ Technology Stack

-   **Frontend:** Next.js 14 (App Router), TypeScript, Tailwind CSS, Shadcn/UI
-   **Backend:** Node.js, Express.js
-   **Database:** MySQL
-   **Image Storage:** Cloudinary
-   **Authentication:** JWT (JSON Web Tokens)
